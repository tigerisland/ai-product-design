
# AI‑Assisted Product Design

This repository documents my exploration of using AI tools (starting with Claude) to support professional product design work.

## Goals
- Develop fluency using AI as a design thinking partner
- Translate vague ideas into clear specs and artifacts
- Better collaborate with engineering through shared language and tools
- Build lightweight prototypes and experiments

## What You’ll Find Here
- Product specs and design explorations
- AI‑assisted critiques and revisions
- Simple prototypes (HTML/CSS)
- Notes on workflows, prompts, and learnings

## Tools
- Claude (primary AI thinking partner)
- GitHub (versioning and documentation)
- VS Code (editing and prototyping)

## Status
This is an active, evolving workspace. Files and structure will change as I learn.
